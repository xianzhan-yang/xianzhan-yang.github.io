Day 1 Linux exercise

Posted on 2024-11-14 In Linux Skill Views:

Question

Write a shell script that does the following:

  1. Checks if a specific directory (e.g., /tmp/test_directory) exists. If it doesn’t, create the directory.
  2. Inside this directory, create 5 text files named file1.txt, file2.txt, etc.
  3. Write “Hello, World!” into each of these files.
  4. Display the contents of all files in the directory.
  5. Clean up by deleting the files and the directory once you’re done.

Bonus: Make the script accept a directory path as an argument, so you can specify a custom directory to work in.

Answer

Explanation

  1. DIR_PATH=”${1:-/tmp/test_directory}” sets the directory path to the first argument provided, or defaults to /tmp/test_directory if no argument is given.
  2. [ ! -d “$DIR_PATH” ] checks if the directory exists; if not, mkdir -p creates it.
  3. A for loop creates files file1.txt to file5.txt and writes “Hello, World!” to each.
  4. cat “$DIR_PATH”/*.txt displays the contents of all text files in the directory.
  5. Finally, rm -rf “$DIR_PATH” deletes the created files and directory.
