Posted on 2024-11-30 In Linux

Question:

You are tasked with maintaining a Linux server and performing regular system administration tasks. Your goal is to:

  1. List all users on the system who have not logged in for the past 30 days and save the list to /home/yourusername/inactive_users.txt.

  2. Check the disk space of the /home directory, and if the disk usage exceeds 90%, send a warning email to admin@yourdomain.com.

  3. Set up log rotation for the /var/log/myapp.log file:

    • Rotate the log file every week.
    • Keep 4 weeks’ worth of logs and delete older logs.
    • Ensure that the log file is compressed after rotation.

  4. Create a script that:

    • Checks the available disk space of the /home directory.
    • Runs the disk space check every hour, and sends a notification email if the usage exceeds 90%.

Hints:

  • Use last or lastlog to check user logins.
  • Use df to check disk space usage.
  • Use mail or sendmail to send email notifications.
  • Use logrotate for managing log rotation.
  • Use cron to schedule tasks.
