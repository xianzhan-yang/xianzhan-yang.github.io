Question:

You are tasked with managing disk space and backups on a Linux server. Perform the following tasks:

  1. Identify the disk usage of all mounted filesystems and save the output to /home/yourusername/disk_usage_report.txt.
  2. Find all files larger than 100MB in the /var directory and its subdirectories and save the list (with file sizes) to /home/yourusername/large_files.txt.
  3. Create a compressed backup of the /etc directory named etc_backup.tar.gz and store it in /home/yourusername/backups/. Ensure the backup includes hidden files.
  4. Set up a cron job to run this backup every day at 3:00 AM.

Hints:

  • Use df to check disk usage.
  • Use find with -size for large file search.
  • Use tar to create a compressed archive.
  • Use crontab to schedule tasks.
