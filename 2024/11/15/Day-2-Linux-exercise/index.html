Day 2 Linux exercise

Posted on 2024-11-15 In Linux Views:

Question

Write a script that:

  1. Lists all files in a specified directory (or the current directory if none is provided) that were modified in the last 7 days.
  2. For each of these files, display the filename, size, and last modification date.
  3. Save this output to a file named recent_files.txt in the same directory as the script.

Bonus: Make the script flexible to accept a custom number of days as an argument, so you can search for files modified within any number of days.

Answer

Explanation

  1. TARGET_DIR=”${1:-.}” sets the target directory to the first argument provided, or defaults to the current directory (.).
  2. DAYS=”${2:-7}” sets the number of days to the second argument or defaults to 7 if not provided.
  3. find “$TARGET_DIR” -type f -mtime -$DAYS finds files modified within the last $DAYS days.
  4. ls -lh –time-style=long-iso lists the files with their sizes and modification dates in a readable format.
  5. awk is used to format the output, showing the filename, size, and modification date, which is then saved to recent_files.txt.
